## Local development

```bash
# clone the repository

# cd in the project directory
$ cd bookworm-light/exampleSite/

# Start local dev server
$ hugo server --themesDir ../..
```
Or Check out [Full Documentation](https://docs.gethugothemes.com/bookworm/?ref=github).
